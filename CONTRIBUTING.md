# Contributing to Simple Interest Calculator

Thank you for your interest in contributing to our project! We are a micro-finance startup dedicated to empowering low-income individuals, and your help makes a big difference.

## Welcome Contributions

We welcome and encourage contributions of all kinds, including:
* **Bug Reports:** If you find an error in the script or documentation, please open an Issue.
* **Bug Fixes:** Submit a Pull Request (PR) to patch any known issues.
* **Documentation Improvements:** Enhancements to the README or code comments are highly appreciated.
* **Enhancements and Ideas:** New features to expand the calculator's capabilities are welcome.

## Submission Process

1. **Fork** the repository to your own GitHub account.
2. **Clone** the project to your local machine.
3. Create a new branch for your feature or fix (`git checkout -b feature-new-cool-thing`).
4. **Commit** your changes with clear messages.
5. **Push** to your fork and open a **Pull Request** against the `main` branch.

Please ensure your contributions align with our Code of Conduct. Happy coding!
