# Simple Interest Calculator

This repository contains a lightweight Bash script designed to calculate simple interest directly from the terminal. 

The project was created as part of our micro-finance startup's technological migration, aiming to provide open, accessible, and fast tools for internal teams.

## Prerequisites

To run this script, you need a Linux-based environment (such as Ubuntu/Lubuntu) or Git Bash on Windows. Please ensure the `bc` utility is installed on your system for decimal mathematical operations.
